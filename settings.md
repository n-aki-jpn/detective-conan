# 名探偵コナン TCG プロトタイプ 設定ファイル

## 言語設定
- **思考言語**: 日本語
- **回答言語**: 日本語
- **ログ言語**: 日本語

## プロジェクト情報
- **プロジェクト名**: Detective Conan TCG Phase 1 Prototype
- **バージョン**: 1.0.0
- **開発フェーズ**: Phase 1 (ターン進行状態マシンとカード移動メカニクス)

## 技術スタック
- **フロントエンド**: Vanilla JavaScript
- **スタイル**: CSS3 (Glassmorphism, Dark Theme)
- **レイアウト**: CSS Grid
- **サーバー**: Python http.server (開発用)

## ファイル構成
```
conan-tcg-prototype/
├── index.html          # メインHTML構造
├── style.css           # プレミアムスタイリング
├── cards.js            # カードデータベース
├── state.js            # ターン進行状態マシン
├── app.js              # イベントハンドラーとUIレンダリング
└── settings.md         # この設定ファイル
```

## ゲームルール
- **ターン構造**: AUTOフェイズ → MAINフェイズ → ENDフェイズ
- **手札プレイ制限**: 1ターンに1回（ネクストヒントで追加可能）
- **コスト**: カードレベル ≤ FILE枚数
- **色ルール**: 事件カードの色と一致（白/赤/黄は例外）
- **名乗り**: 登場ターンは推理/アクション不可

## 開発メモ
- ホットシートモード（2人プレイヤー同画面）
- サンドボックスツールでテスト可能
- ドラッグ＆ドロップ対応
- ロングプレス（400ms）でズーム表示

## 引き継ぎ書更新ルール
- エラー修正を除いて毎回作業終了時に必ず引き継ぎ書（HANDOVER.md）を更新してください
- 引き継ぎ書には以下の内容を含めること：
  - プロジェクトの概要
  - 実装済みの機能
  - 直近の修正内容
  - 次に確認すべきこと
  - ファイル構成
  - 重要な技術的詳細
  - ゲームルール
  - サンドボックスツール
